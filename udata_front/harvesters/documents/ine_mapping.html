Lógica de Processamento Otimizada

O harvester utiliza um mecanismo de Cache em Memória para mitigar a lentidão dos servidores do INE.

Mecanismo de Cache

O processo de harvest foi consolidado em uma única requisição de rede de larga escala:

  • Discovery com Cache: Ao ler o catálogo, o harvester mapeia todos os indicadores para um objeto interno.
  • Processamento Zero-Network: O detalhamento de cada dataset consome os dados do cache, eliminando milhares de requisições HTTPS redundantes.
  • Fallback: Caso um dataset não esteja no catálogo inicial, é realizado um fetch individual resiliente.

Tratamento de Strings

  • Limpeza: Remoção de avisos de browser e ruído fora da tag XML.
  • CDATA: Extração robusta de conteúdos em blocos CDATA.
